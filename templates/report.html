Descrição do Modelo

O modelo foi treinado utilizando um pipeline que inclui as seguintes etapas:

  1. Carregamento dos Dados: Os dados foram carregados de um arquivo CSV.
  2. Definição das Features e Target: Foram definidas as features 'experience_level', 'employment_type', 'job_title', e 'employee_residence' como variáveis independentes e 'salary_in_usd' como a variável dependente.
  3. Divisão dos Dados: Os dados foram divididos em conjuntos de treino e teste usando uma proporção de 80/20.
  4. Transformação das Variáveis: As variáveis categóricas foram transformadas em vetores one-hot usando o OneHotEncoder.
  5. Construção do Pipeline: Um pipeline foi criado para incluir a pré-processamento e o modelo preditivo.
  6. Grid Search: Foi realizado um Grid Search com validação cruzada para encontrar os melhores hiperparâmetros para diferentes regressores.
  7. Treinamento do Modelo: O modelo com os melhores parâmetros foi treinado nos dados de treino.
  8. Avaliação do Modelo: O modelo foi avaliado usando métricas como MSE, R² Score, e MAE.
